Class CalendarRound

A combination of 260-day cycles and the Haab cycle. This class should not be instantiated directly, and should be accessed through getCalendarRound.

see

{getCalendarRound}

example

let cr = new CalendarRound(4, "Ajaw", 8, "Kumk'u");
